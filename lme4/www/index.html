To cite lme4, use the following 2 references: 

Jeroen C.L. Ooms (2009). yeroon.net/lme4: A web interface for the R package lme4. 
Version 0.1. http://www.yeroon.net/lme4

Douglas Bates and Martin Maechler (2009). lme4: Linear mixed-effects models using S4 classes. 
R package version 0.999375-31. http://CRAN.Rproject.org/package=lme4
